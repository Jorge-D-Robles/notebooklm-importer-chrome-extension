How to Use NotebookLM Importer

  1. Open a NotebookLM notebook in a browser tab.
  2. Copy the URL of that notebook (it should look like `https://notebooklm.google.com/notebook/...`).
  3. In the extension popup, paste the URL into the "Notebook URL" box and click Save. This is a one-time setup per notebook.
  4. To add a single website, navigate to that page and click Add Current Tab in the extension popup.
  5. To add all your open websites at once, click Add All Tabs.

You should see a loading spinner, followed by a green checkmark for success or a red X for failure.
